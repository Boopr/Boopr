About the Project

Techonolgies used in this project

This project can be found on github with instructions on deployment here. This project was the result of our final capstone project worked on at Codeup, a coding bootcamp for Full-Stack Java Web Development. The goal was to have a responsive website using rest controllers and AJAX. A lot of this website has custom built javascript classes and modules to generate a lot of the UI. Most of it allowed us to have quick control on editing reduntent commands for creating elements and dealing with events.

The Future

Going forward with this project, we aim to use Vue.JS with VueX and seperate the frontend from the backend. This will allow the app to be more flexable with our goals of making an Android app in the coming time since Java is currently under our belts now. We will also implement JWT for security to make the api easier to use for other frontends.

New features to add and lots more are in store that will make this more exciting and cool place to post your cute dog photos. Stay tuned to the project as there is more to come!
